#How it works

DAVP is a decentralized protocol for publishing and verifying cryptographic proofs. Truth is determined by cryptography, distribution is handled by peer-to-peer networking, and no node is ever trusted with authority.

Asset Proofs

Immutable proof objects containing a BLAKE3 hash, creator Ed25519 signature, UTC timestamp, and metadata (type, AI flag, tags, parent verification_id). Signed and globally unique.

Decentralized Network

Peer-to-peer mesh with gossip replication. A Common Network Tracker (CNT) bootstraps verified nodes; after bootstrap, the protocol runs without any central component.

Verification

Anyone can verify locally: recompute the hash, check the Ed25519 signature, and validate metadata. Optional issuer certificates can be queried off-chain for organizational trust.

Monetization

Protocol usage is always free. Businesses pay for bulk registration, certified/legal timestamps, enterprise APIs, and trusted node operation. Payments are out-of-band; the protocol remains open.

Security

Adoption
